# Install ShotCode Scanner on Mobile

The web version works perfectly on mobile and can be installed as an app!

## Option 1: Install as PWA (Recommended - Works Best!)

### Android (Chrome/Edge):
1. Open https://sentore1.github.io/00code/ in Chrome
2. Tap the menu (⋮) → "Install app" or "Add to Home Screen"
3. Tap "Install"
4. The app icon will appear on your home screen
5. Open it like any other app - works offline!

### iPhone (Safari):
1. Open https://sentore1.github.io/00code/ in Safari
2. Tap the Share button (□↑)
3. Scroll down and tap "Add to Home Screen"
4. Tap "Add"
5. The app icon will appear on your home screen

## Option 2: Just Use the Website

Simply bookmark https://sentore1.github.io/00code/ and use it directly in your browser. It works perfectly for scanning!

## Why PWA is Better Than Native App:

✅ **Works immediately** - no build/install process
✅ **Always up-to-date** - automatically gets latest version
✅ **Smaller size** - ~1MB vs 20MB+ for native app
✅ **Camera access** - full camera API support
✅ **Offline capable** - works without internet after first load
✅ **Cross-platform** - same code works on Android & iOS
✅ **No app store** - no approval process needed

## Troubleshooting:

### Camera not working:
1. Check browser permissions (Settings → Site Settings → Camera)
2. Make sure you're using HTTPS (required for camera)
3. Try refreshing the page

### Can't install as app:
- Make sure you're using Chrome (Android) or Safari (iOS)
- Some browsers don't support PWA installation
- You can still use it as a website - works the same!

### Scanning not accurate:
- Ensure good lighting
- Hold phone steady
- Center the code in frame
- Try different distances
- Make sure code is not blurry
